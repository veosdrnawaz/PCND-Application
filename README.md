# Pak Currency Note Distribution (PCND) Application

A modern, responsive web application that uses Machine Learning (Scikit-learn) and a FastAPI backend to predict the optimal distribution of Pakistani currency note denominations (Rs. 1000, Rs. 500, Rs. 100, Rs. 50, Rs. 20, Rs. 10) for a given amount.

## Project Architecture

- **Machine Learning**: RandomForestRegressor with MultiOutputRegressor in Scikit-learn.
- **Backend**: FastAPI (Python) serving predictions and API.
- **Frontend**: Premium Material Design 3 (HTML, Vanilla CSS, JS) styled in Pakistan Green and Emerald Green.
